# Week 5 - PySpark Data Cleaning and Aggregation

## Objective

The objective of this assignment is to understand Spark DataFrames and perform data cleaning, filtering, transformations and aggregation operations using PySpark.

## Dataset Used

Dataset: Shopping Trends Dataset

Total Rows: 3900

Some important columns used:
- Customer ID
- Age
- Category
- Purchase Amount (USD)
- Location
- Subscription Status

## Tasks Performed

- Removed duplicate records
- Handled null values using na.drop() and na.fill()
- Applied filtering conditions on age and subscription status
- Performed groupBy operations
- Calculated count, sum, average, minimum and maximum values
- Understood shuffle and wide transformations
- Modified schema using casting and renaming
- Created a complete data processing pipeline

## Technologies Used

- Google Colab
- PySpark
- Python

## Files in this Repository

- week5_pyspark_assignment.ipynb

## Conclusion

This assignment helped in understanding Spark DataFrames, data cleaning techniques, aggregation functions and transformation operations. Spark provides better performance than traditional MapReduce because of its in-memory processing capabilities.
